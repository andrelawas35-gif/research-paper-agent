# ADR 0024: Note Signals Are Provisional

Personal Notes may produce Candidate Signals, but they will not automatically become durable User Model preferences or interests. A Note Card can suggest a preference, recurring workflow, or research interest, but promotion into the User Model requires an explicit memory request or repeated evidence over time.

## Considered Options

- Promote every note-derived signal automatically: highly adaptive, but likely to overfit from rough thoughts.
- Keep notes entirely separate from personalization: clean, but misses useful adaptation opportunities.
- Treat note-derived signals as provisional: selected because it matches the existing adaptive grill memory policy while letting notes influence near-term behavior.
