# Modes Live Inside Unified Workspaces

The PWA exposes Now, Chat, Library, Work, People, Compass, Reviews, and Settings as stable Workspaces rather than placing every Agent Mode in top-level navigation. Capture, Retrieve, Source, Synthesis, Builder, Grill, Tutor, Reflect, Relationship, Taste, Review, Writing, Artifact, Admin, Mentor, Regulation, and Values behavior is inferred within those Workspaces and may change without forcing navigation. This preserves the complete PKM mode taxonomy for agent behavior while keeping the interface organized around recognizable user objects and tasks.
