# ADR 0021: Notes in the Concept Graph

Personal Notes and Note Cards can introduce or reinforce concepts in the Concept Graph, while keeping their provenance separate from paper evidence. This changes the graph from an interest-to-paper-concept structure into a typed source graph where User Interests, Papers, Personal Notes, Note Cards, Concepts, and prerequisite hints can connect through shared concept vocabulary.

## Considered Options

- Attach notes only to existing Paper Concepts: safer vocabulary, but prevents the user from capturing ideas before papers mention them.
- Keep a separate notes graph: cleaner boundaries, but loses the main value of relating personal thinking to research evidence.
- Extend the Concept Graph with note provenance: selected because it lets papers and notes meet through Concepts without confusing Personal Notes with evidence-backed Papers.
