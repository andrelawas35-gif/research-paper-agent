# ADR 0052: Hybrid Learning Control

The Self-Learning Knowledge Loop will use hybrid learning control. Weak always-on learning may update Candidate Signals, interaction logs, Tutor Progress, and Concept Graph usage weights; durable controls remain explicit through Profile Mode, explicit note capture, Knowledge Self-Audit corrections, and Builder/Self-Improvement proposals.

This lets the agent learn from ordinary use while keeping durable preferences, identity-shaping claims, and self-modifying changes visible and user-controlled.
