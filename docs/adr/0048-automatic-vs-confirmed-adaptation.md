# ADR 0048: Automatic vs Confirmed Adaptation

The Self-Learning Knowledge Loop will allow automatic weak/local updates but require confirmation for durable identity-level behavior changes. Candidate Signals, Tutor Progress, interaction logs, and graph weights from explicit notes or use may update automatically; durable preferences, avoidances, major behavior rules, concept merges, and code/self-modification proposals require explicit user confirmation.

This gives the agent self-learning momentum without over-promoting weak inferences into long-term behavior.
