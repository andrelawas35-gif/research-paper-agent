# Separate Regulation, Values, Cognitive Support, and Orientation Modules

The current agent will add four modules: Emotional Regulation owns sessions, safety workflow, reality checks, outcomes, patterns, and its store; Values Compass owns values discovery, orientation hierarchy, lifecycle, reminders, and its store; Cognitive Support owns dimensions, tactics, outcomes, defaults, and its store; Personal Orientation owns policy, filtering, ranking, budgets, and snapshot assembly. Root-agent tools remain thin wrappers, domain modules do not read one another's files, and each supplies policy-safe orientation candidates through its interface. This creates additional modules but prevents `user_profile.py` from becoming an unsafe catch-all identity database and gives personalization one explicit integration seam.
