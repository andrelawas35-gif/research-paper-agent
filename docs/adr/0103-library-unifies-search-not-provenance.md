# Library Unifies Search, Not Provenance

Library provides one cross-PKM search experience while every result remains visibly typed as external evidence, user-authored knowledge, model inference, confirmed decision, provisional candidate, Concept, or generated Work Object. Filters cover type, Concept, source, date, confirmation, sensitivity, and related Goal or project, and result cards expose source-specific provenance. This improves discovery without flattening Personal Notes into citations or generated summaries into source evidence.
