# Memory Candidates Expire and Declines Are Suppressed

Weak inferred Memory Candidates expire after 30 days, explicit Owner statements remain reviewable for 90 days, and sensitive or identity-shaping candidates expire after 30 days unless deliberately deferred. Deferral supplies a chosen return date or allows resurfacing only when directly relevant. Declined candidates leave only a minimal suppression fingerprint and are never usable as personal memory; they may be proposed again only when materially new evidence is attached and shown. Expired candidates leave active review and retrieval without affecting accepted durable records, and the Memory Inbox may quietly summarize expiry without demanding action. This prevents stale inference from becoming shadow knowledge while preserving deliberate, accepted memory.
