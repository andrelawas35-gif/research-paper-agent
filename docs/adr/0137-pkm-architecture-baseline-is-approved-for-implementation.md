# PKM Architecture Baseline Is Approved for Implementation

ADRs 0073 through 0136 form the approved implementation baseline for the whole Personal Knowledge Manager, including its unified Modes and workspaces, Regulation, Values Compass, ADHD-aware Cognitive Support, reviewed memory adaptation, UI and privacy surfaces, hybrid retrieval, GPT model routing, bounded costs and context, single-VM production architecture, scheduling, recovery, and release evidence. Implementation may refine reversible details but may not silently contradict an accepted decision; new evidence changes the baseline through a superseding ADR. Work follows the six safety-first Tracer-Bullet Milestones in ADR 0134 and initially targets the existing 1-OCPU, 6-GB Oracle A1 VM. A 4-OCPU, 24-GB resize is optional headroom, while a separate agent, PostgreSQL, standalone vector database, distributed queue, multi-VM high availability, and local LLM remain deferred until measured thresholds justify their complexity.
