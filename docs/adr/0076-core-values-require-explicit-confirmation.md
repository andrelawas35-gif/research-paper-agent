# Core Values Require Explicit Confirmation

The Values Compass may derive Candidate Values from historical conversations, Personal Notes, goals, study choices, and repeated behavior, but only the user can promote one to a Core Value. Every candidate retains supporting and conflicting Value Evidence, and isolated emotional conversations cannot establish identity-level conclusions. This sacrifices fully passive personalization to prevent interests, habits, or temporary states from being misrepresented as the user's enduring values.
