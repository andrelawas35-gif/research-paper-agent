# ADR 0033: Conservative Note Card Extraction

Personal Notes will use conservative Note Card extraction: one to five high-confidence reusable ideas by default, not a card for every sentence. This keeps the Notes Store useful for retrieval and graph linking without flooding the Concept Graph with low-value fragments.

## Considered Options

- Extract every sentence: easy and comprehensive, but noisy.
- Extract only one summary card: clean, but loses multi-idea notes.
- Extract a few high-confidence cards: selected because it balances recall, graph quality, and note readability.
