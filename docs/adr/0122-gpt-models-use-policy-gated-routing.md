# GPT Models Use Policy-Gated Routing

The initial production model provider is OpenAI. GPT-5 mini is the default model for daily PKM conversation, retrieval synthesis, study support, values grounding, and model-assisted Regulation sessions. GPT-5 is available only through explicit user choice or a code-owned escalation policy for work whose ambiguity or consequence justifies its added cost; emotional activation alone is not an escalation signal. Cheaper models may later perform non-authoritative background extraction or classification only after passing task-specific evaluation, while Regulation outages use the Deterministic Regulation Protocol rather than an unqualified cheap-model substitute. Model identifiers remain configuration behind the model-provider Replacement Seam, and no model may enter Regulation Mode until it passes the private evaluation set for structured output, rule adherence, reassurance resistance, coercion avoidance, calibrated uncertainty, Cognitive Support delivery, values grounding, latency, and cost.
