# ADR 0049: Knowledge Self-Audit

The agent will expose a Knowledge Self-Audit view that separates confirmed user facts from inferred guesses and demonstrated knowledge state. The audit should show explicit preferences and avoidances, candidate reasoning patterns, note-derived signals, mastered and weak concepts, strongest/stale/rejected graph concepts, and merge suggestions.

This keeps the Self-Learning Knowledge Loop inspectable and correctable as the agent adapts to the user's quirks, intelligence, interests, and knowledge level.
