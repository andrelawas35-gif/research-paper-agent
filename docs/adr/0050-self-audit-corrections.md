# ADR 0050: Self-Audit Corrections

The Knowledge Self-Audit will support correction actions so the user can steer the Self-Learning Knowledge Loop. The initial correction surface should confirm a Candidate Signal into a durable item, reject a Candidate Signal so it stops resurfacing, downgrade an over-promoted preference, and reject a Concept so it is suppressed in graph ranking.

This makes the audit view a control surface rather than a passive report.
