# Local-Only Means Deterministic on the A1 VM

On the current 1-OCPU, 6-GB Oracle A1 VM, local-only processing means deterministic application behavior rather than a locally hosted conversational LLM. It supports explicit dates, commitments, names, tags, and preferences; secret and identifier removal; FTS5 retrieval; duplicate and conflict detection; typed-record parsing; manual memory governance; and the Deterministic Regulation Protocol. It does not claim reliable nuanced inference of values, emotional patterns, relationship dynamics, or purpose from free-form history; those require separately consented GPT processing or Owner-authored records. A local model may be evaluated later on suitable hardware, but it may not share production resources until workload tests demonstrate safe latency and memory margins.
