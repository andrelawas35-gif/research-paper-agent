# ADR 0026: Note-Guided Questioning

Adaptive Grill and Tutor Mode will use Personal Notes and Note Cards as ranking signals and prompt context when choosing the next question. Notes can make a concept more personally relevant, but they do not replace cited paper evidence for teaching, grading, or source-backed claims.

## Considered Options

- Ignore notes during questioning: keeps behavior simple, but misses the user's current thinking.
- Let notes drive questions and grading directly: highly personal, but confuses reflections with evidence.
- Use notes for ranking and context only: selected because it personalizes the learning loop while preserving evidence-grounded teaching.
