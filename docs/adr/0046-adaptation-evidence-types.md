# ADR 0046: Adaptation Evidence Types

The Self-Learning Knowledge Loop will distinguish explicit stated preferences, inferred reasoning patterns, and demonstrated knowledge state. Explicit preferences can become durable User Model entries immediately; reasoning patterns begin as Candidate Signals and require repetition or confirmation; knowledge state is tracked through Tutor Progress, grill answers, corrections, and repeated prompts.

This lets the agent adapt to the user's quirks, intelligence, and knowledge level without over-promoting weak signals into durable behavior rules.
