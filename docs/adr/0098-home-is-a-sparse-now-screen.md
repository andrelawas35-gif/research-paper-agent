# Home Is a Sparse Now Screen

The PWA opens to a Now Screen containing at most one relevant orientation, one active Commitment, one pending follow-up, two primary actions, and one optional suggestion, with empty sections hidden and every personalized item explainable, dismissible, snoozable, or reducible through feedback. Sensitive Relationship Context, distress frequency, unfinished-goal counts, and broad analytics stay off the default surface. This limits overview density so ADHD-aware orientation reduces choice burden rather than turning the user's personal history into a control panel.
