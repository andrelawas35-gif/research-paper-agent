# Separate Purpose, Values, Principles, Goals, and Commitments

The Values Compass models Purpose, Core Values, Personal Principles, Goals, and Commitments as distinct layers with different scopes and rates of change. It also preserves Value Tensions instead of forcing a permanent global ranking, allowing the agent to recommend contextual trade-offs. This prevents temporary goals from being mistaken for identity, makes reminders actionable, and lets the user revise tactics without implying that their deeper values have changed.
